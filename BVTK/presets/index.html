Download the batch script
Below you can download the batch script to automatically create animations via the terminal. The batch.zip folder contains three files, among which BVTK.sh, that is the shell script to run. Don't forget to run 'chmod +x' both on BVTK.sh and BVTK.config. If you have troubles with the linux interpreter you may try to rewrite the script with nano.
